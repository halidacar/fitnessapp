# Fitness Application
- It is designed for beginners in sports. It is a mobile application designed for both Android and iOS devices, easy to use with initial gestures.
------------------------------------------------------------------------------------------------------------------------------------

# Create

- npx create-expo-app fitnessapp

- cd fitnessapp
- npx expo start

------------------------------------------------------------------------------------------------------------------------------------

# Run

- yarn/npm install
- npm run ios/android

# Dependecies 

- npx expo install react-native-screens react-native-safe-area-context
  // safe area context
- npm install react-native-screens react-native-safe-area-context
- npx pod-install ios // for ios devices

   # Async-Storage

  With npm:
  
  - npm install @react-native-async-storage/async-storage
  
  With Yarn:
  
  - yarn add @react-native-async-storage/async-storage
  
  With Expo CLI:
  
  - npx expo install @react-native-async-storage/async-storage
